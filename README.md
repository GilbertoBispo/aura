# 💪🗿 Projeto: **Aura**

Uma rede social fictícia para aqueles que querem "farmar aura". A ideia é ser uma rede social gamificada e inspirada no meme, com sistema de níveis como em um RPG.

Comecei a desenvolver esse projeto com o intuito de aprender React e Tailwind na prática, além de integrar o frontend feito com essas tecnologias com um backend feito em Node + Express.

## 🛠️ Tecnologias utilizadas até o momento:

### Backend
- Node.js v25
- Express v5
### Frontend
- React v19
- Tailwind v4
- React Router v7.14.0
- Lucide v1.7.0

## 📝 Próximas features:

- Criação de um banco de dados remoto utilizando Neon e Drizzle ORM
- Feed Dinâmico: Implementação de props e renderização de listas para exibir múltiplos posts a partir de uma base de dados simulada.
- Página de Perfil Gamificada: Exibição detalhada do nível do usuário, conquistas desbloqueadas e "Aura Total" acumulada.
- Engine de Engajamento: Sistema de feedback visual ao curtir/descurtir (animações de +1 ou -1 Aura subindo na tela).
